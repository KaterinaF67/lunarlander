Симулятор посадки лунного модуля с реалистичной физикой и автопилотом на базе PD-регуляторов. Проект написан на C++ с использованием wxWidgets для графического интерфейса.
### Физическая модель
- Реалистичная физика движения с учётом массы, момента инерции и расхода топлива
- 4 двигателя: 2 основных (нижние) и 2 боковых для коррекции
- Влияние ветра на траекторию полёта
### Автопилот (AutoPilot.h)
Трёхфазная система автоматической посадки:
- 1. **POLY (Полиномиальная траектория)**
   - Построение квинтической траектории до целевой зоны
   - Адаптивное планирование времени полёта
   - PD-регулятор для отслеживания траектории

2. **BRAKE (Торможение)**
   - Активное гашение горизонтальной скорости
   - Автоматический выбор безопасной точки посадки
   - PD-регулятор с уменьшенными коэффициентами

3. **VERTICAL (Вертикальная посадка)**
   - Выравнивание по нормали к поверхности
   - Плавное снижение до касания 
   - Строгий контроль вертикальной скорости
  
### Радарная система
- 7 лучей веерного сканирования (конус 28°)
- Детекция опасных зон (высота < 35 м)
- Фильтрация ложных срабатываний на резком рельефе
- Визуализация лучей с цветовой индикацией опасности

### Графический интерфейс (lunar2.cpp)
- Рисование произвольного рельефа мышью
- Визуализация траектории полёта
- Отображение безопасных зон посадки 
- Отображение целевой точки автопилота
- HUD с параметрами полёта в реальном времени
- Управление скоростью симуляции (1x-10x)
